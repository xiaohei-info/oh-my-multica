# Solution Design Business Flow Diagramming

## Overview

This skill owns **解决方案设计阶段的业务流程图 artifact**.

## Canonical Solution-Design Slice (Preserved)

| 业务流程有哪些？ _业务问题与需求拆解，不同的角色与对应的需求问题_ | 给出业务流程图 |

## What This Artifact Must Answer

- which roles participate
- what steps happen in which order
- what the business handoffs are

## Verification Checklist

- [ ] Roles are explicit
- [ ] Steps and order are explicit
- [ ] Handoffs are explicit
- [ ] Artifact stays at business-flow level
