oh-my-multica Overall Architecture

A production delivery control layer between callers, platform-neutral engine ports, and execution systems.

Delivery Control Layer

  • • Turns design, acceptance, and DAG artifacts into executable delivery state.
  • • Uses deterministic loops, gates, and explicit recovery decisions to drive convergence.

Platform-neutral Engine Ports

  • • WorkItemStore owns work-item and evidence semantics.
  • • AgentRuntime owns bounded task dispatch without exposing platform CLIs.

Execution and Feedback

  • • Multica supplies the collaboration and runtime foundation through adapters.
  • • Coding Agents, Git, Pull Requests, and CI return delivery facts to the gates.
